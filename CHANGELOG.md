# Changelog

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-11-19

### Добавлено
- Независимый Vue компонент Media2 для управления медиа файлами
- Поддержка работы без токенов авторизации
- Управление папками и файлами
- Загрузка файлов с прогрессом
- Drag & Drop загрузка
- Поиск и фильтрация файлов
- Пагинация
- Просмотр изображений и видео через lightbox
- Перемещение файлов между папками
- Корзина с восстановлением файлов
- Настраиваемый базовый URL API через props
- Полная документация и примеры использования

### Изменено
- Убрана зависимость от токенов авторизации
- Упрощена структура API запросов
- Удален ImageEditor (не включен в пакет)

### Технические детали
- Vue 3 Composition API
- Независимые утилиты API без авторизации
- Laravel Service Provider для интеграции
- Поддержка Laravel 10.x и 11.x
